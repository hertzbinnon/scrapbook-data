回答1:

As you can see from the console output the issue is that the three inputs have different SAR values: 0:1 for the second input vs 40:33 for the first and third inputs.

You can set the corect SAR value using the setsar filter, eg:

 [1:v:0]setsar=sar=40/33[1v0sar]

then pass [1v0sar] to the concat filter along with the other video streams.
