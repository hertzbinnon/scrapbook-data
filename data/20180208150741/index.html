Force a CBR in x265 using FFMpeg

up vote 0 down vote favorite

I'm currently conducting an experiment, where I am comparing the performance of H.264 and H.265. I have managed to get a CBR on H.264 (x264) using -nal-hrd cbr. However, this does not work for x265. Does anyone know a work around?

Below is a line of code I have tried in FFMpeg:

ffmpeg -i 'Clip 1.mov' -c:v libx265 -pix_fmt yuv444p -b:v 500k -minrate 500k -maxrate 500k -bufsize 500k -nal-hrd cbr 'Clip 1_cbr_500.mp4'

Thanks, Rhys.
